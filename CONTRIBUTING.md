# Contribuyendo a Docai 🚀

¡Gracias por interesarte en mejorar este proyecto! Para mantener el orden y la calidad del código, por favor sigue estas instrucciones:

## 1. ¿Cómo puedo contribuir?

### Reportar Errores (Bugs)
Si encuentras un error, abre un "Issue" en el repositorio detallando:
- Pasos para reproducir el error.
- Comportamiento esperado vs. comportamiento real.
- Capturas de pantalla si el error es visual.

### Sugerir Mejoras
¡Las ideas son bienvenidas! Abre un "Issue" con la etiqueta `enhancement` explicando por qué esa función sería útil.

## 2. Proceso de Pull Request (PR)
1. Haz un **Fork** del proyecto.
2. Crea una rama para tu mejora: `git checkout -b mejora/nueva-funcion`.
3. Haz tus cambios y asegúrate de que el servidor de FastAPI arranca correctamente.
4. Haz un commit claro: `git commit -m "Añadida conexión con dataset X"`.
5. Envía un **Pull Request** a la rama `main`.

## 3. Código de Conducta
Sé amable y respetuoso con los demás colaboradores. Como dice la comunidad Open Source: "Build Community, Be Community".
