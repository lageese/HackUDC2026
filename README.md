## 🛠️ Tecnologías y Atribuciones

Este proyecto ha sido posible gracias al uso de las siguientes herramientas de código abierto y plataformas:

* **[Denodo SDK](https://www.denodo.com/):** Utilizado para la virtualización de datos y el acceso eficiente a los datasets de Pokémon y Digimon.
* **[Ollama](https://ollama.com/):** Motor de inferencia local para ejecutar el modelo de lenguaje que procesa las consultas.
* **[FastAPI](https://fastapi.tiangolo.com/):** Framework de Python utilizado para construir la API del backend (Licencia MIT).
